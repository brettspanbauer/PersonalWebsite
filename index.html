SQL Experience

Our in-house web applications were attached to a SQL server. This server was used to store reports and other significant data that was input from the web application. I learned how the Django Framework leverages a server to simplify the processing of pulling data out, and putting data in. Along with this, I also learned to embed SQL code into SQL Server Reporting Services from Microsoft (SSRS). I was given the task of finding a new way to generate invoices for audits, as our current server at the time was old and outdated, which caused a security risk. All the data from the old server was pushed into a SQL database, and my job was to find a way to get that data out when a coworker or auditor needed it. Using SSRS, I developed an application where a coworker could enter the site, input some fields to find the invoice they were looking for, and get an invoice out that they could save and do whatever they needed with it. This was accomplished by embedding SQL queries into the reports that I made.
